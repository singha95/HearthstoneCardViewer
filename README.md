# HearthstoneCardViewer
A app to view and create decks in Hearthstrone by Blizzard

Run Server: 
python manage.py runserver

Update Database: 
python manage.py makemigrations
python manage.py migrate 

Create New App: 
python manage.py startapp <app name>
